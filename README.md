# Taby - 标签页管理扩展

一个简单而强大的 Chrome 标签页管理扩展。

## 功能特点

- 🗂️ 标签页分组管理
- 🔄 拖拽排序
- 💾 本地数据持久化
- ☁️ GitHub Gist 同步
- 📱 响应式布局

## 安装扩展

1. 打开 Chrome 扩展页面 (`chrome://extensions/`)
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择项目 `dist` 目录

## 数据同步

支持通过 GitHub Gist 进行数据同步：

1. 创建 GitHub Personal Access Token
2. 在扩展中配置 Token
3. 点击同步按钮即可进行数据备份/恢复
